CitiBike Ridership 2018 Analysis

Interactive Dashboard with Tableau

This report analyzes rental data from New York City's CitiBike program for Quarter 1 of 2018, including station popularity, rider demographics, and trip statistics. Data was collected from the CitiBike website and visualized using Tableau. The per-station Start vs. End Stats were queried using Python and Pandas.

Tools used: Tableau, Python, Pandas
