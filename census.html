U.S. Census - Demographics vs. Health

Interactive Dashboard

The purpose of this project was to compare demographic and health data by state using data from the 2014 US Census. Data was selected from the US Census and CDC Behavioral Risk Factor websites, cleaned using Excel, and saved in CSV format. The CSV was then used to drive an interactive scatterplot, with multiple axis selection, using D3.
